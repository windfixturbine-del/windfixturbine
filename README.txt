PLIKI:
- index.html
- styles.css
- app.js
- assets/logo.png

WDROŻENIE:
1) Rozpakuj ZIP.
2) Na hostingu przejdź do folderu strony: public_html / www / htdocs.
3) Zrób kopię starej wersji: index.html -> index_old.html (opcjonalnie).
4) Wgraj WSZYSTKIE pliki z paczki:
   - index.html
   - styles.css
   - app.js
   - folder assets/ (z logo.png)
5) Odśwież stronę: Ctrl+F5 / Cmd+Shift+R

Język można wymusić linkiem:
?lang=pl | ?lang=de | ?lang=en

ANIMACJE:
- Subtelne "reveal on scroll" (IntersectionObserver)
- Ikona turbiny w badge delikatnie się obraca
- Wspiera prefers-reduced-motion (dostępność)
